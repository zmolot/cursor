Основні команди Git:

  • git config--global user.name blabla
  • git config --global user.email blabal@bla
  • git init
  • git status
  • git add
  • git commit
  • git log
  • git checkout
  • git branch
  • git push
  • git pull
  • git merge
  • git clone

.gitignore - файл, де вказуються папки та файли, що не будуть відслідковуватись і завантажуватись (ігноруються) у віддалений репозиторій.

Advanced команди Git:

  • git revert
  • git reset
  • git rebase
  • git cherry-pick
  • git stash
